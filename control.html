A large part of my research involves the development of control software and code to increase the information that can be obtained from an experiment. Some activities:

  • Developed a control interface for both motors used to drive SHIVA (300 and 15 kW), and set-up control of new axial load piston (see here)
  • MATHSdaq, a MATLAB gui interface to acquire data using TiePie handyscopes
  • Qdaq, a MATLAB interface to acquire attenuation measurements and perform live data reduction
